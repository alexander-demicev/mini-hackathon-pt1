# Factorial

You want to calculate the factorial of an integer N Factorial refer as N! and calculate the following formula: 

N! = 1 * 2 * 3 * ... * (N-1) * N and 0! = 1.

The same is valid recurrent relation: N! = (N-1)! * N

In a single line of input file INPUT. TXT recorded one returns a nonnegative integer N (N < 1000).

In the output file OUTPUT. TXT FILE you want to display a single integer is N!.
