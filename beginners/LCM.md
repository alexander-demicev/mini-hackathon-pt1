# LCM
## Complexity 75%

Write a program defining the Least Common Multiple (LCM) of the numbers a and b.

INPUT contains two natural numbers A and B separated by a space

OUTPUT a single integer - the LCM of the numbers A and B.

| INPUT                             | OUTPUT                 |
|-----------------------------------|------------------------|
| 36 27	                            | 108                    |
| 39 65	                            | 195                    |
