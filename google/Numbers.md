# Numbers
## Complexity 40%

Find the smallest positive integer Q such that the product of its digits is equal to the given number N.

INPUT contains one integer N

TXT number Q, if such a number does not exist, you should output -1.



| INPUT                             | OUTPUT                 |
|-----------------------------------|------------------------|
| 10                                | 25                     |
| 13                                | -1                     |
| 90                                | 259                    |
